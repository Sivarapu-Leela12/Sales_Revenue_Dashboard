#  Sales Revenue Dashboard

 # Project Overview
The Sales Revenue Dashboard project analyzes sales data to generate meaningful business insights using data cleaning, analysis, and visualization techniques.
It helps understand revenue trends, profit distribution, and category-wise performance through reports and dashboards.

# Objective
To clean, analyze, and visualize sales data and create an interactive dashboard for better business decision-making.

# Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Power BI
- Excel / CSV

# Project Workflow

# 1. Data Collection
- Imported sales dataset from CSV/Excel file.

# 2. Data Cleaning
- Handled missing values
- Removed duplicates
- Corrected data types and formatting

# 3. Data Analysis
- Grouped sales by category and region
- Calculated total revenue and profit
- Generated summary reports

# 4. Data Visualization
- Bar charts for category-wise sales
- Line charts for sales trends over time
- Pie charts for profit distribution

# 5. Dashboard Creation
- Built interactive Power BI dashboard
- Added filters for dynamic analysis
- Displayed KPIs (Sales, Profit, Orders)
  
#  Key Insights
- Identified top-performing sales categories
- Analyzed profit trends over time
- Improved understanding of business performance

# Outcome
This project provides automated sales reporting and interactive dashboards that help in data-driven decision-making.
